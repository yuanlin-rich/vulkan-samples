= 基础硬件加速光线追踪（Basic hardware accelerated ray tracing）

ifdef::site-gen-antora[]
TIP: 本示例源码位于 https://github.com/KhronosGroup/Vulkan-Samples/tree/main/samples/extensions/ray_tracing_basic[Khronos Vulkan Samples GitHub 仓库]。
endif::[]

*扩展*：`VK_KHR_ray_tracing_pipeline`、`VK_KHR_acceleration_structure`

使用跨厂商的光线追踪扩展渲染基础场景。展示如何设置光线追踪所需的数据结构，包括几何体的 BLAS/TLAS、着色器绑定表（SBT）以及包含射线生成、命中、未命中着色器组的光线追踪管线。光线分派后将结果拷贝到交换链图像。
