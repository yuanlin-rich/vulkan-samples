= 基本光线查询（Basic ray queries）

ifdef::site-gen-antora[]
TIP: 本示例源码位于 https://github.com/KhronosGroup/Vulkan-Samples/tree/main/samples/extensions/ray_queries[Khronos Vulkan Samples GitHub 仓库]。
endif::[]

*扩展*：`VK_KHR_ray_query`、`VK_KHR_acceleration_structure`

使用光线查询扩展渲染 Sponza 场景，展示如何设置光线查询所需的数据结构，包括几何体的 BLAS/TLAS 以及标准顶点/片段着色器管线。片段着色器通过光线查询动态投射阴影。
