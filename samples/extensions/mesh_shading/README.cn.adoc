= 网格着色（Mesh shading）

ifdef::site-gen-antora[]
TIP: 本示例源码位于 https://github.com/KhronosGroup/Vulkan-Samples/tree/main/samples/extensions/mesh_shading[Khronos Vulkan Samples GitHub 仓库]。

本示例演示最基础的 mesh shading 用法：在网格着色器中直接生成一个三角形；不使用顶点着色器，仅包含网格着色器与片段着色器。
