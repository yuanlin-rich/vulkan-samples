= 推送描述符（Push Descriptors, HPP）

ifdef::site-gen-antora[]
TIP: 本示例源码位于 https://github.com/KhronosGroup/Vulkan-Samples/tree/main/samples/extensions/hpp_push_descriptors[Khronos Vulkan Samples GitHub 仓库]。
endif::[]

*扩展*：https://www.khronos.org/registry/vulkan/specs/latest/html/vkspec.html#VK_KHR_push_descriptor[`VK_KHR_push_descriptor`]

推送描述符将“推送常量”的理念应用到“描述符集”。本示例不为每个对象创建描述符集，而是在“录制命令缓冲”时直接传递描述符。
