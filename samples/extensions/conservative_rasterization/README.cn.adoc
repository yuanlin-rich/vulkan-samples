= 保守光栅化（Conservative Rasterization）

ifdef::site-gen-antora[]
TIP: 本示例源码位于 https://github.com/KhronosGroup/Vulkan-Samples/tree/main/samples/extensions/conservative_rasterization[Khronos Vulkan Samples GitHub 仓库]。
endif::[]

*扩展*：https://www.khronos.org/registry/vulkan/specs/latest/html/vkspec.html#VK_EXT_conservative_rasterization[`VK_EXT_conservative_rasterization`]

使用保守光栅化改变片段生成方式：启用“过估计（overestimation）”后，为每个被几何体触碰到的像素生成片段，而不仅仅是完全覆盖的像素。
