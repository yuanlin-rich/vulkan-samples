= OpenGL 互操作（OpenGL interoperability）

ifdef::site-gen-antora[]
TIP: 本示例源码位于 https://github.com/KhronosGroup/Vulkan-Samples/tree/main/samples/extensions/open_gl_interop[Khronos Vulkan Samples GitHub 仓库]。
endif::[]

*扩展*：`VK_KHR_external_memory`、`VK_KHR_external_semaphore`

用 OpenGL 渲染程序化图像，并将该内容整合进 Vulkan 场景。演示在 OpenGL 与 Vulkan 中共享纹理的同一底层内存，以及通过共享信号量与屏障同步两个 API。
