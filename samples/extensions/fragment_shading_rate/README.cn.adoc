= 片段着色率（Fragment shading rate）

ifdef::site-gen-antora[]
TIP: 本示例源码位于 https://github.com/KhronosGroup/Vulkan-Samples/tree/main/samples/extensions/fragment_shading_rate[Khronos Vulkan Samples GitHub 仓库]。
endif::[]

*扩展*：https://www.khronos.org/registry/vulkan/specs/latest/man/html/VK_KHR_fragment_shading_rate.html[`VK_KHR_fragment_shading_rate`]

使用专用帧缓冲附件控制不同帧缓冲区域的片段着色率；可显式控制每个被片段覆盖的像素的片段着色器调用次数，例如用于“注视点渲染（foveated rendering）”。
